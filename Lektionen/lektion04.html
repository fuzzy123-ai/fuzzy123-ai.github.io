Bedingungen in Python

In Python können Bedingungen mit dem if-Statement ausgedrückt werden. Das Statement wird verwendet, um einen Block von Code auszuführen, wenn eine Bedingung erfüllt ist. Wenn die Bedingung nicht erfüllt ist, wird der Code-Block übersprungen.

Beispiel

Das folgende Beispiel zeigt die Verwendung von if und else in Python:

    x = 5
    if x > 10:
        print("x ist größer als 10")
    else:
        print("x ist kleiner oder gleich 10")

Der Code prüft, ob die Variable x größer als 10 ist. Da dies nicht der Fall ist, wird der Code-Block unterhalb von else ausgeführt und "x ist kleiner oder gleich 10" wird ausgegeben.

Übungsaufgaben

Übe die Verwendung von Bedingungen in Python, indem du die folgenden Aufgaben löst:

  1. Erstelle eine Variable y mit dem Wert 8. Schreibe eine Bedingung, die prüft, ob y größer als 10 ist. Wenn dies der Fall ist, gib "y ist größer als 10" aus, ansonsten gib "y ist kleiner oder gleich 10" aus.
  2. Erstelle eine Variable z mit dem Wert -3. Schreibe eine Bedingung, die prüft, ob z kleiner als 0 ist. Wenn dies der Fall ist, gib "z ist negativ" aus, ansonsten gib "z ist nicht negativ" aus.
