Dateien einbinden in Python

Um Dateien in Python einzubinden, gibt es verschiedene Möglichkeiten, je nachdem welche Art von Datei es ist und was man damit machen möchte.

less Copy code

Textdateien einlesen

Um eine Textdatei einzulesen, kann man die Funktion open() verwenden. Hierbei wird die Datei im Lesemodus geöffnet und mit der Methode read() ausgelesen.

      
        with open('meine_datei.txt', 'r') as f:
            inhalt = f.read()

Bilder einbinden

Um Bilder einzubinden, kann man das Modul PIL verwenden. Hierbei wird das Bild mit der Methode Image.open() geöffnet und kann dann weiterverarbeitet werden.

      
        from PIL import Image

        img = Image.open('mein_bild.jpg')
        img.show()

CSV-Dateien einlesen

Um CSV-Dateien einzulesen, kann man das Modul csv verwenden. Hierbei wird die Datei geöffnet und mit einer Schleife kann man durch die einzelnen Zeilen iterieren und die Daten weiterverarbeiten.

      
        import csv

        with open('meine_csv_datei.csv', 'r') as f:
            csv_reader = csv.reader(f)

            for row in csv_reader:
                print(row)

    summe = 0
    for line in file:
        try:
         zahl = int(line)
         summe += zahl

         continue
    print("Die Summe der Zahlen in der Datei beträgt:", summe)
