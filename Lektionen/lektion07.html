Funktionen in Python

Funktionen in Python sind wiederverwendbare Codeblöcke, die einen bestimmten Zweck erfüllen.
Sie werden in der Regel geschrieben, um Code zu organisieren, die Lesbarkeit zu verbessern und die Wiederverwendbarkeit von Code zu fördern.

Einfaches Beispiel

  
    def hallo_welt():
        print("Hallo Welt")

Die Funktion wird aufgerufen, indem man einfach den Namen der Funktion eingibt:

  
    hallo_welt()

Komplizierteres Beispiel

  
    def fibonacci(n):
        if n <= 1:
            return n
        else:
            return (fibonacci(n-1) + fibonacci(n-2))

Die Funktion wird aufgerufen, indem man den Parameter 'n' übergibt:

  
    print(fibonacci(5))

Das Ergebnis lautet: 5 (da die 5. Zahl in der Fibonacci-Folge 5 ist)

Wichtige Dinge, die man bei Funktionen beachten sollte

  • Funktionen sollten einen klaren Zweck erfüllen und gut benannt sein.
  • Funktionen sollten so geschrieben sein, dass sie wiederverwendbar sind.
  • Funktionen sollten so kurz wie möglich sein.
  • Funktionen sollten eine klare Dokumentation haben, die beschreibt, was sie tun und welche Parameter sie benötigen.

Übungsaufgaben zum Erstellen von Funktionen in Python

Aufgabe 1

Schreibe eine Funktion, die den größten Wert in einer Liste von Zahlen zurückgibt.

Lösung:

Größter Wert in einer Liste

Die Funktion groesster_wert() nimmt eine Liste von Zahlen und gibt den größten Wert zurück.

        
          def groesster_wert(liste: list) -> float:
              """
              Diese Funktion nimmt eine Liste von Zahlen und gibt den größten Wert zurück.
    
              Argumente:
              liste -- Eine Liste von Zahlen.
    
              Rückgabe:
              Der größte Wert in der Liste.
              """
              # Setze den Anfangswert des größten Werts auf den ersten Wert in der Liste.
              groesster = liste[0]
    
              # Durchlaufe alle Werte in der Liste und aktualisiere den größten Wert.
              for wert in liste:
                  if wert > groesster:
                      groesster = wert
    
              # Gib den größten Wert zurück.
              return groesster

Beispiel:

        
          zahlen = [5, 8, 2, 10, 3]
          groesster_zahl = groesster_wert(zahlen)
          print("Der größte Wert in der Liste ist:", groesster_zahl)

Die Syntax "-> float" am Ende der Zeile gibt an, dass die Funktion einen Fließkommawert zurückgibt.
Auch diese Information ist optional, da Python die Rückgabeart dynamisch erkennt.
Es kann jedoch hilfreich sein, die Rückgabeart für andere Entwickler oder für die Code-Dokumentation zu definieren.
