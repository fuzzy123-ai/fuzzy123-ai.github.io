Variablen in Python

Zahlen

Python hat zwei grundlegende Arten von Zahlen:

  • Ganze Zahlen (int): z.B. 42
            zahl = 42
  • Gleitkommazahlen oder Dezimalzahlen (float): z.B. 3.14
            zahl = 42.88452

Booleans

Booleans repräsentieren Wahrheitswerte und haben nur zwei mögliche Werte:

                
            bool = True
  • True
  • False

Strings

Strings repräsentieren Text und werden in Python in Anführungszeichen gesetzt:

    
            str = "hello world"
  • Einfache Anführungszeichen: 'Hallo Welt'
  • Doppelte Anführungszeichen: "Hallo Welt"

Listen

Listen sind eine geordnete Sammlung von Elementen. Sie können Elemente hinzufügen, entfernen und ändern:

          
            my_list = [1, 2, 3]
            my_list.append(4)  # Fügt 4 am Ende der Liste hinzu
            my_list[1] = 5    # Ändert das zweite Element auf 5

Dictionaries

Dictionaries sind eine Sammlung von Schlüssel-Wert-Paaren. Sie ermöglichen es, schnell auf Elemente zuzugreifen:

          
            my_dict = {'name': 'Max', 'alter': 27}
            my_dict['name']   # Gibt 'Max' zurück
            my_dict['alter']  # Gibt 27 zurück

Aufgabe

Schreibe ein Python Programm, dass den Input für Radius und Höhe eines Zylinders abfragt und daraus das Volumen berechnet. Einheiten sollten angegeben werden

Lösung

Hier ist die Lösung:

                
    PI = 3.14159

    radius = float(input("Bitte gib den Radius des Zylinders in Metern ein: "))
    hoehe = float(input("Bitte gib die Höhe des Zylinders in Metern ein: "))

    grundflaeche = PI * radius ** 2
    volumen = grundflaeche * hoehe

    print("Das Volumen des Zylinders beträgt", round(volumen, 2), "Kubikmeter.")

Das Ergebnis wird mit round() auf zwei Nachkommastellen gerundet und mit einer Einheit ausgegeben.
