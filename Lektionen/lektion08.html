Benutzeranmeldung in Python erstellen


Aufgabe:

Schreibe ein Python-Programm, das ein Fenster mit einem Benutzernamen- und Passwort-Eingabefeld sowie einem Absenden-Button enthält.
Wenn der Benutzername 'Benutzer' und das Passwort '12345678' sind, gib eine Erfolgsmeldung aus.
Wenn die Anmeldeinformationen ungültig sind, gib eine Fehlermeldung aus."

Vorgehen:

  • Importiere tkinter
  • Erstelle ein Fenster mit TkInter:
  •           root = tk.Tk()            //das Hauptfenster wird definiert, Groß- und Kleinschreibung beachten!
          root.title("Login")       //Name vom Fenster
          root.geometry("300x150")  //300x150 ist die Größe in Pixeln
  • statt root, könntest du es auch hauptfenster nennen
  • Erstelle "label" (Text)
  • Erstelle "entry" (Eingabe), mit show="*"können wir die Texteingabe verschleiern
  • Erstelle "button" (Knopf), der button soll den command=check_login ausführen
  • Nicht vergessen! Jedes Element muss gepackt werden mit button.pack()
  • Starte die mainloop mit root.mainloop()
  • Definiere eine Funktion: check_login
  • Mit dem Befehl element.get() können wir eine Eingabe auslesen
  • Wir benutzen from tkinter import messagebox um die Möglichkeit zu bekomen Popup Windows zu erzeugen
  • Mesasgebox.showinfo("Titel", "Text")
  • statt Info können wir auch eine Error box erzeugen

   import tkinter as tk
    from tkinter import messagebox

    def check_login():
        username = username_entry.get()
        password = password_entry.get()

        if username == "benutzer" and password == "12345678":
            messagebox.showinfo("Success", "Login successful!")
        else:
            messagebox.showerror("Error", "Invalid login credentials.")

    root = tk.Tk()
    root.title("Login")
    root.geometry("300x150")

    username_label = tk.Label(root, text="Username:")
    username_label.pack()

    username_entry = tk.Entry(root)
    username_entry.pack()

    password_label = tk.Label(root, text="Password:")
    password_label.pack()

    password_entry = tk.Entry(root, show="*")
    password_entry.pack()

    submit_button = tk.Button(root, text="Submit", command=check_login)
    submit_button.pack()

    root.mainloop()
