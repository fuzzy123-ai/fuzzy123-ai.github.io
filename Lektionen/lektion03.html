Schleifen in Python

For-Schleifen

Eine For-Schleife wird verwendet, um über eine bestimmte Sequenz von Elementen zu iterieren, wie zum Beispiel eine Liste oder einen String:


for element in liste:
    print(element)

Dabei wird die Schleife für jedes Element in der Liste ausgeführt und das Element in der Variablen 'element' gespeichert.

While-Schleifen

Eine While-Schleife wird so lange ausgeführt, wie eine bestimmte Bedingung erfüllt ist:


while bedingung:
    # Schleifeninhalt

Dabei wird der Schleifeninhalt ausgeführt, solange die Bedingung wahr ist.

Beispielaufgaben

Übe das Schleifen in Python mit den folgenden Beispielaufgaben:

Aufgabe 1

Erstelle eine For-Schleife, die die Zahlen von 1 bis 10 ausgibt.

Aufgabe 2

Erstelle eine While-Schleife, die alle ungeraden Zahlen von 1 bis 20 ausgibt.

Aufgabe 1


            for zahl in range(1, 11):
            print(zahl)

Aufgabe 2


        zahl = 1
        while zahl <= 20:
            if zahl % 2 != 0:
                print(zahl)
            zahl += 1
